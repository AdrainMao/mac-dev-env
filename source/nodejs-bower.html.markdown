> **Dependencies:** [Node.js](/nodejs/)

**Bower** is a front-end package manager.


### Install

Install Bower globally using [NPM](https://npmjs.org/).

	npm install -g bower


### Verify the Installation

Verify that you have successfully installed **Bower**.

	bower --version
