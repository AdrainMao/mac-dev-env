### Install

Install Ruby on Rails using [RubyGems](http://rubygems.org/).

	gem install rails

Rebuild the `rbenv` shim binaries.

	rbenv rehash


### Bugs

The `rails` command reports as missing in some cases after installation. Opening a new **Terminal** window resolves this.
